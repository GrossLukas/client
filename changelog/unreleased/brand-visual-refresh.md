Enhancement: Clean visual refresh in the owncloud.online brand colors

The client now uses the owncloud.online brand turquoise (#00e2bb, the
app-icon color) as its accent throughout the main window instead of the
operating system's blue:

- The active tab in the top bar (account, Activity, Settings) gets a
  turquoise underline and a subtle turquoise tint - so it is always
  obvious where you are.
- Account chips without an avatar picture use the brand colors: calm
  teal, full brand turquoise for the currently selected account.
- Selected rows in the folder list and the folder browser highlight in
  translucent brand turquoise. The tint keeps the normal text color, so
  it stays readable in both the light and the dark theme.

The logo, icons and all layouts are unchanged - just cleaner, calmer
and consistently in brand colors.
